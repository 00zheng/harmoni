HOW TO DOWNLOAD A CHROME EXTENSION

Download the extension from GitHub
Open the GitHub repo page.
Click Code → Download ZIP.
Unzip it somewhere you can find again (Desktop is fine).
Find the correct extension folder (important)
When you unzip, you often get a folder like:
timer-extension-main/
Open Chrome’s Extensions page
Open Chrome.
Go to: chrome://extensions/
Turn on the toggle in the top-right: Developer mode
Load it as an unpacked extension
Click Load unpacked
Select the folder that contains manifest.json
Click Select Folder
If it loads correctly, you’ll see the extension appear in your list.
